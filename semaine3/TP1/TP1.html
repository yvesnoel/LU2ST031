TP 1#

Introduction#

Bienvenue sur JupyterHub. Vous êtes dans le notebook du TP n°1. Nous avons vu qu’un notebook est constitué de cellules. Ces cellules peuvent contenir du code Python ou bien du text Markdown (mémo Markdown ici). Dans les notebooks de cours, vous trouverez une alternance de textes (introduction, énoncés, explications) et de code Python (exemples, code à compléter ou modifier, ou code à créer complètement. Certaines cellules ne sont pas modifiables, d’autres le sont si vous le désirez et d’autres encore DOIVENT être complétez (et seront évaluées). Lorsque vous aurez terminé ce TP, vous le soumettrez grâce à Jupyterhub (cf procédure ici). Votre enseignant pourra alors le récupérer, le corriger si besoin et faire des remarques éventuelles.

Objectifs

A l’issue de ce TP, vous serez capable de :

  • reconnaître le type d’une expression

  • évaluer une expression comme le ferait Python en appliquant les règles de priorité

  • traduire une expression mathématique relativement simple en une expression Python

  • affecter le résultat d’une expression à une variable

  • utiliser le contenu d’une variable dans une expression

  • reconnaître le type d’une variable déjà existante

  • modifier le contenu d’une variable déjà existante
